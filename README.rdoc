= redblog

RedBlog is a simple blog plugin for redmine 1.2+ it adds a blog entry in the top menu. User must login to redmine in order to post to blog

= Installation

* extract RedBlog to  vendors/plugin
* rake db:migrate_plugins RAILS_ENV=production
* Done
